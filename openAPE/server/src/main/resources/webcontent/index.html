Welcome to OpenAPE


This is the OpenAPE project. OpenAPe is the open Accessibility Personalisation Extension. It is an light wight, REST based infrastructure to exchange context information to provide personalised user interfaces. OpenAPE is the reference implementation of ISO/IEC 24752-8.

If you want to benefit from the OpenAPE infrastructure you need to create an account.
