Welcome to OpenAPE


This is the OpenAPE project. OpenAPe is the open Accessibility Personalisation Extension. It is an light wight, REST based infrastructure to exchange context information to provide personalised user interfaces. the reference implementation of ISO/IEC 24752-8.
